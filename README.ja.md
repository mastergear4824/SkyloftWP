Midjourney TVのリアルタイムAI生成動画を壁紙として表示し、ローカルライブラリに自動保存してオフラインでも視聴できるmacOSアプリです。

> 💡 **注意**: このアプリはMidjourney TV用に設計されましたが、**他のストリーミングビデオサービス**にも接続して、Macのダイナミック壁紙として使用できます。

## ✨ 主な機能

### 🎬 ストリーミング接続
- **バックグラウンドMidjourney TV接続** - WebViewによるリアルタイム動画ストリーミング
- **カスタムビデオソース** - Midjourney TV以外のストリーミングURLを追加可能
- **バッファモード（デフォルト）** - ToS準拠のため保存なしで直接再生
- **オプションの自動保存** - ユーザーが希望すればライブラリに保存可能
- **ネットワーク状態監視** - 接続切断時の自動再接続

### 📚 ライブラリ管理
- **SQLiteデータベース** - 動画メタデータの永続保存
- **自動サムネイル生成** - AVAssetImageGeneratorでサムネイル抽出
- **お気に入りと再生回数追跡**
- **動画非表示（いいねしない）** - 不要な動画をローテーションから除外
- **スマートクリーンアップ** - 制限超過時に古い動画を自動削除

### 🖥️ 壁紙再生
- **デスクトップレベルウィンドウ** - アイコンの下に表示される真の壁紙
- **マルチモニターサポート** - 各モニターで独立した動画再生
- **連続再生** - ライブラリから順次/ループ再生
- **オーバーレイ設定** - 透明度、明るさ、彩度、ブラーの調整

### 🌏 多言語対応
- English (en)
- 한국어 (ko)
- 日本語 (ja)

## 📖 使用ガイド

### 1️⃣ ライブラリ

<p align="center">
  <img src="capture/Library.png" width="700" alt="Library">
</p>

保存されたAI生成動画をグリッドレイアウトで管理します。

- **サムネイルプレビュー**: 各動画の代表画像を表示
- **動画再生**: クリックして壁紙に設定
- **右クリックメニュー**: 
  - 壁紙に設定
  - プロンプトをコピー
  - Midjourneyで開く
  - Finderで表示
  - 非表示 / 削除

---

### 2️⃣ ストリーミング設定

<p align="center">
  <img src="capture/Streaming.png" width="700" alt="Streaming">
</p>

Midjourney TV接続と自動保存設定を構成します。

| オプション | 説明 |
|------------|------|
| **接続状態** | ストリーミング接続状態（接続済み/未接続） |
| **ビデオソース** | Midjourney TV（デフォルト）またはカスタムURL追加 |
| **ライブラリに保存** | 有効にすると新しい動画を自動でライブラリに保存（**デフォルトOFF**） |
| **最大動画数** | ライブラリに保持する最大動画数（超過時に古い動画を自動削除） |

> 🛡️ **デフォルト: バッファモード** - サービス利用規約準拠のため、自動保存はデフォルトで無効です。動画は保存せずに直接再生されます。ユーザーは希望すれば保存を有効にできます。

> 💡 **ヒント**: ストリーミングが接続されると、アプリがバックグラウンドでビデオソースを監視し、新しい動画が表示されると自動でダウンロードします。

> 🔗 **カスタムソース**: 連続ビデオコンテンツを提供する任意のストリーミングビデオサービスURLを追加できます。これにより、Midjourney TV以外の様々なビデオ壁紙ソースにアプリを活用できます。

### 3️⃣ ディスプレイ設定

<p align="center">
  <img src="capture/Display.png" width="700" alt="Display">
</p>

壁紙の視覚効果を調整します。

**プリセット**
| プリセット | 効果 |
|------------|------|
| Default | オリジナルのまま |
| Subtle | やや暗く |
| Dim | 暗く |
| Ambient | 雰囲気のある |
| Focus | フォーカスモード（ブラー追加） |
| Vivid | 色彩強化 |
| Cinema | シネマティック |
| Neon | ネオン効果 |
| Dreamy | 夢のような |
| Night | ナイトモード |
| Warm | 暖かいトーン |
| Retro | レトロスタイル |
| Cool | クールなトーン |
| Soft | ソフトに |

**手動調整**
- **透明度**: 壁紙の不透明度 (0~100%)
- **明るさ**: 明るさ調整 (-100% ~ +100%)
- **彩度**: 色の彩度 (0~200%)
- **ブラー**: ブラー効果 (0~50)

## 🚀 ビルド

### Xcodeでビルド

1. `AIStreamWallpaper.xcodeproj`をXcodeで開く
2. `AIStreamWallpaper`スキームを選択
3. Product > Build (⌘B)

### コマンドラインでビルド

```bash
xcodebuild -project AIStreamWallpaper.xcodeproj \
           -scheme AIStreamWallpaper \
           -configuration Release \
           -derivedDataPath build
```

ビルドされたアプリの場所: `build/Build/Products/Release/AIStreamWallpaper.app`

### DMG配布版の作成

```bash
./build-dmg.sh
```

出力: `dist/AIStreamWallpaper.dmg`

```
│   └── {uuid}.mp4

└── Buffer/              # バッファモード一時ファイル
```

## 🚀 自動起動設定

ログイン時の起動はLaunchAgentで実装されています：

```
~/Library/LaunchAgents/com.aistreamwallpaper.plist
```

メニューバーまたは設定で「ログイン時に起動」を有効にすると、このファイルが自動生成されます。
